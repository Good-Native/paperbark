# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its code
of conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

This Code applies within all project spaces — issues, pull requests,
discussions, and any other forum used by the project — and also applies
when an individual is officially representing the project in public
spaces.

## Enforcement

Concerns may be reported to the project maintainers at
**support@goodnative.co**. All reports will be reviewed and
investigated promptly and fairly. Maintainers are obligated to respect
the privacy and security of anyone who reports an incident.
